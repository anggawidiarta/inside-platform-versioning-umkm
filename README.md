# Inside Platform Landing Page

# Requirements & Stack:

- NodeJS LTS 16+
- Yarn 1.22.10+
- React 18.2.0
- NextJS 13.4.6
- Antd 5.11.1
- Typescript 5.1.3+

# Running application development mode

- Copy file `.env.example` rename to `.env` then setting the configuration
- Run `yarn install`
- Run `yarn dev`

# Running application production mode

- Copy file `.env.example` rename to `.env` then setting the configuration
- Run `yarn install`
- Run `yarn start`

# Build

- Copy file `.env.example` rename to `.env` then setting the configuration
- Run `yarn install`
- Run `yarn build`
